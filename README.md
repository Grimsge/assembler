# Ассемблер для Учебной Виртуальной Машины (УВМ)

## Описание проекта
Ассемблер, преобразующий программы из человекочитаемого JSON-формата в машинный код для УВМ. Создает файл с записью результата ассемблирования.

## Формат файла программы
- Входной файл: программы для ассемблера должны быть в формате **JSON** и иметь расширение `.json`.
- Выходной файл: будет создан, рекомендуется расширение `.bin` или `.uvm`.
## Использование ассемблера

### Синтаксис команды:
```bash
python assembler.py <входной_файл> <выходной_файл> [--test]
```
аргумент для режима тестирования опционален
## Поддерживаемые команды:

load_const - Загрузка константы в регистр :
```
{
  "command": "load_const",
  "register": <номер_регистра (0-31)>,
  "value": <константа (0-536870911)>
}
```
read_mem - Чтение из памяти в регистр :
```
{
  "command": "read_mem",
  "register": <номер_регистра (0-31)>,
  "memory_address": <адрес_памяти (0-1048575)>
}
```
write_mem - Запись регистра в память :
```
{
  "command": "write_mem", 
  "value_register": <регистр_со_значением (0-31)>,
  "address_register": <регистр_с_адресом (0-31)>
}
```
sgn_operation - Унарная операция sgn() :
```
{
  "command": "sgn_operation",
  "base_register": <базовый_регистр (0-31)>,
  "target_register": <регистр_назначения (0-31)>,
  "offset": <смещение (0-4095)>
}
```
## Пример содержимого файла:
```
{
  "command": "load_const",
  "register": 6,
  "value": 495
}
```
